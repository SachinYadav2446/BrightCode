## 📦 Project Structure

```
BrightCode/
├── client/                 # React frontend (Vite + React)
│   ├── src/
│   │   ├── components/    # Reusable UI components
│   │   ├── pages/         # Page-level components
│   │   ├── context/       # React Context providers
│   │   ├── services/      # API service layer
│   │   └── data/          # Static data and configs
│   └── public/            # Static assets
├── server/                # Express backend
│   ├── utils/             # Utility modules
│   ├── migrations/        # SQL migration files
│   ├── ai_test_cases/     # AI-generated test case storage
│   └── .env               # Environment config (not committed)
├── docs/                  # Documentation (you are here)
│   ├── INDEX.md           # This file
│   ├── NAVIGATION_GUIDE.md
│   ├── guides/            # Setup and configuration guides
│   ├── features/          # Feature documentation
│   ├── troubleshooting/   # Problem-solving guides
│   └── project_updates/   # Change log and update notes
├── archive/               # Historical docs (older phases)
├── test_scripts/          # Standalone test scripts
└── .kiro/specs/           # Kiro feature specifications
```
